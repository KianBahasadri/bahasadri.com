# Video Call Control Center - User Features

**User-facing feature description from the user's perspective. This document describes what users can do and what the feature provides.**

## Overview

A page that allows users to start and join a video call.

## Key Features

### Call Session Management

Users can create new video call sessions and join.

## User Workflows

### Create and join a Video Call Session

**Steps**:
1. Navigate to the video call center page
2. Click "Create New Call" button
3. System places the user in a new call

### Join existing call session

**Steps**:
1. view the ongoing calls in the main page
2. click a call
3. system places user in the call

## User Capabilities

- Create new video call sessions
- join video call sessions

## Use Cases

### video calling people

video call with someone
